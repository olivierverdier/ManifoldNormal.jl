ManifoldNormal.jl

Noise Models

A general interface for noises on manifolds.

using ManifoldsBase
using ManifoldNormal
σ = 2.0 # the standard deviation
M = ManifoldsBase.DefaultManifold(2)
# choose an isotropic noise for this example
noise = IsotropicNoise(M, σ)
x = rand(M)
""
# output
""
sample_space(noise)
# output
DefaultManifold(2; field = ℝ)
Σ = get_covariance_at(noise, x, DefaultOrthonormalBasis()) 
# output
2×2 PDMats.ScalMat{Float64}:
 4.0  0.0
 0.0  4.0
import Random
rng = Random.default_rng()
y = noise(rng, x) # another point of M
using Manifolds
is_point(M, y)
# output
true

ManifoldNormal.scaled_distance — Method
scaled_distance(D::ProjLogNormal, x)

Start with a ProjLogNormal(μ,Σ) distribution with mean $μ$ lying on the sample manifold $M$, and covariance $Σ$ in the Lie algebra $\mathfrak{g}$ of the group $G$. The infinitesimal action of the group $G$ on the manifold $M$ gives rise to the linear map $P \colon \mathfrak{g} \to T_{μ}M$, which in turns gives the projected covariance $PΣP^*$ on the tangent space $T_{μ}M$.

One then measures the scaled distance from $μ$ to $x$ from $v := \log(μ, x)$ with the formula

\[\frac{\sqrt{v^T (PΣP^*)^{-1} v}}{\sqrt{n}}\]

(where $v$ is regarded as a column matrix here) and $n$ is the dimension of the sample manifold $M$.
